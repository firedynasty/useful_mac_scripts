### Using `tree`

After installation, you can run the `tree` command in your terminal to display the directory structure. Here are some basic ways to use `tree`:

- To display the tree for the current directory:
  ```bash
  tree
  ```

- To display the tree for a specific directory:
  ```bash
  tree /path/to/directory
  ```

- To limit the display to a certain number of levels of directories deep:
  ```bash
  tree -L 2
  ```

- To include files in the listing (by default, `tree` lists files as well):
  ```bash
  tree -f
  ```

- To ignore a specific directory or pattern:
  ```bash
  tree -I 'pattern'
  ```

- To display only directories and not files:
  ```bash
  tree -d
  ```

These are just a few examples of how `tree` can be used. For more options and information, you can check the man page by running `man tree` in your terminal after installation.
